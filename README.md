# Детектор занятости столика по видео

Прототип системы для анализа занятости столиков в ресторане на основе видео. Система детектирует, когда столик пуст или занят, и вычисляет время между освобождением стола и следующим подходом человека.

## 🚀 Возможности

- Детекция людей в области столика с помощью YOLOv8
- Автоматическое отслеживание состояния: пусто/занято
- Фиксация событий с временными метками
- Расчет среднего времени между уходом и следующим подходом
- Визуализация результата с цветовой индикацией
- Экспорт отчета в текстовый файл

## 📋 Требования

- Python 3.8+
- OpenCV
- Pandas
- Ultralytics YOLO
- NumPy

## 🔧 Установка и запуск

### Windows

```bash
# Клонирование репозитория
git clone https://github.com/Invis874/table-occupancy-detector.git
cd table-occupancy-detector

# Создание и активация виртуального окружения
python -m venv venv
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py --video video1.mp4
```

Но можно просто запустить файл `run.bash` если у Вас Windows, не забудьте выделить область стола и нажать "Enter".
Отчет ищите в файле `report_*.txt` в командной строке пропишеться под каким точным именем он был зарегестрирован


## 📹 Виды запуска программы 

### Базовый запуск
Выбираете область стола сами, после чего жмете "Enter"
```bash
python main.py --video video1.mp4
```

### С указанием выходного файла
```bash
python main.py --video video1.mp4 --output result.mp4
```

### С ручным указанием области столика

Кординаты можно посмотреть в файле `Кординаты столов.txt`

```bash
python main.py --video video1.mp4 --roi 100,200,300,250
```
